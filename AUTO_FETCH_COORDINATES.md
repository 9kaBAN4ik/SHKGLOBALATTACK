# Автоматическое получение координат деревень из Stronghold Kingdoms

## Обзор

Теперь Discord бот поддерживает **автоматическое получение координат деревень** напрямую с игрового сервера Stronghold Kingdoms, без необходимости ручного ввода.

## Команды

### 1. `/time all` - Автоматический режим (НОВОЕ!)

```
/time all fromids: 16107;74469 targetid: 83239
```

**Как это работает:**
- Бот автоматически запрашивает координаты деревень с игрового сервера
- Не требуется ручной ввод координат
- Работает для ЛЮБЫХ деревень на сервере (не только ваших)
- Возвращает таблицу с временем атак для 6 множителей скорости

**Пример ответа:**
```
📡 Получение координат деревень с игрового сервера...

⚔️ Расчет времени атак
Цель: Деревня #83239

+------------+-----------+-----------+-----------+-----------+-----------+-----------+
| Village_id | Attack x1 | Attack x2 | Attack x3 | Attack x4 | Attack x5 | Attack x6 |
+------------+-----------+-----------+-----------+-----------+-----------+-----------+
|   16107    |   6m:5s   |   3m:2s   |   2m:2s   |   1m:31s  |   1m:13s  |    1m:1   |
|   74469    |   6m:16s  |   3m:8s   |   2m:5s   |   1m:34s  |   1m:15s  |   1m:3s   |
+------------+-----------+-----------+-----------+-----------+-----------+-----------+

✅ Координаты получены с игрового сервера | Скорость капитана: 15 tiles/hour
```

### 2. `/time attack` - Ручной режим (кэш)

```
/time attack fromids: 16107;74469 targetid: 83239
```

**Как это работает:**
- Использует ранее сохраненные координаты из кэша
- Быстрее, но требует предварительного добавления координат
- Подходит для часто используемых деревень

**Если деревни не найдены:**
```
❌ Деревни не найдены в кэше: 16107, 74469
Используйте `/time all` для автоматического получения координат с сервера
Или добавьте координаты вручную: `!coords <village_id> <x> <y> <название>`
```

### 3. `!coords` - Добавить координаты вручную

```
!coords 16107 500 600 My Village
```

Добавляет координаты деревни в локальный кэш для быстрого доступа.

## Технические детали

### Архитектура

```
Discord Bot (bot.py)
    ↓
Proxy Server (proxy_server.py)  
    ↓
GameServerClient.cs (C#)
    ↓
Stronghold Kingdoms Game Server
```

### Файлы

1. **analysis/bot.py** - Discord бот
   - Команда `/time all` - автоматическое получение
   - Команда `/time attack` - использование кэша
   - Команда `!coords` - ручное добавление

2. **proxy_server.py** - REST API прокси
   - `/api/auto-calculate-attacks` - автоматический endpoint
   - `/api/calculate-attacks` - endpoint с использованием кэша
   - `/api/fetch-villages` - получение координат с сервера

3. **GameServerClient.cs** - C# клиент для игры
   - Подключение к игровому серверу через .NET Remoting
   - Получение списка всех деревень на сервере
   - Возвращает координаты в JSON формате

4. **auto_village_fetcher.py** - Утилита для массовой загрузки
   - Получает координаты всех/конкретных деревень
   - Автоматически кэширует в proxy_server

### API Endpoints

#### POST /api/auto-calculate-attacks

Автоматически получает координаты (если отсутствуют) и рассчитывает время атак.

**Request:**
```json
{
  "targetVillageId": 83239,
  "attackingVillageIds": [16107, 74469]
}
```

**Response (Success):**
```json
{
  "target_village_id": 83239,
  "target_village_name": "Village_83239",
  "attacks": [
    {
      "village_id": 16107,
      "village_name": "Village_16107",
      "distance": 141.42,
      "travel_time": "9h:25m:40s",
      "from_coords": "(100, 200)",
      "to_coords": "(200, 300)"
    }
  ]
}
```

**Response (Missing Villages):**
```json
{
  "error": "Villages not found in cache",
  "missing_villages": [16107, 74469],
  "message": "Please add village coordinates using !coords command or the game server integration"
}
```

#### POST /api/fetch-villages

Получить координаты конкретных деревень с игрового сервера.

**Request:**
```json
{
  "village_ids": [16107, 74469, 83239]
}
```

**Response:**
```json
{
  "message": "Fetched and cached 3 villages",
  "cached_villages": 3
}
```

## Использование

### Запуск системы

1. **Запустить прокси-сервер:**
```bash
python proxy_server.py
```

2. **Запустить Discord бота:**
```bash
cd analysis
python bot.py
```

### Примеры использования

**Пример 1: Автоматический расчет**
```
Пользователь: /time all fromids: 16107;74469 targetid: 83239
Бот: 📡 Получение координат деревень с игрового сервера...
Бот: [таблица с результатами]
```

**Пример 2: Использование кэша**
```
Пользователь: /time attack fromids: 16107;74469 targetid: 83239
Бот: [таблица с результатами из кэша]
```

**Пример 3: Деревни не найдены**
```
Пользователь: /time attack fromids: 99999 targetid: 83239
Бот: ❌ Деревни не найдены в кэше: 99999

```
Пользователь: !coords 16107 500 600 My Attack Village
Бот: ✅ Координаты деревни 16107 установлены: (500, 600) и отправлены в прокси

## Преимущества

✅ **Не требуется ручной ввод** - координаты получаются автоматически
✅ **Работает для любых деревень** - не только ваших, но и чужих
✅ **Быстрый расчет** - результат за несколько секунд
✅ **Кэширование** - повторные запросы работают мгновенно
✅ **Гибкость** - можно использовать как автоматический, так и ручной режим

## Ограничения

⚠️ **Требуется авторизация** - нужны учетные данные в .env файле
⚠️ **Доступ к серверу** - должен быть доступ к игровому серверу
⚠️ **Timeout** - запрос к игровому серверу может занять до 30 секунд

## Устранение неполадок

### Проблема: "Не удалось подключиться к прокси-серверу"
**Решение:** Убедитесь что `proxy_server.py` запущен на localhost:5000

### Проблема: "Превышено время ожидания ответа от игрового сервера"
**Решение:** 
- Проверьте подключение к интернету
- Убедитесь что игровой сервер доступен
- Попробуйте еще раз позже

### Проблема: "Не удалось получить координаты деревень"
**Решение:**
- Проверьте правильность village ID
- Убедитесь что деревни существуют на сервере
- Используйте `!coords` для ручного добавления координат
