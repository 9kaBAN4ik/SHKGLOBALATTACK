# Итоги работы

## ✅ Что сделано

### 1. Discord Бот (Python)
- Команда `!time` - расчет времени атак
- Команда `!coords` - добавление координат деревень
- Команда `!addvillages` - быстрое добавление тестовых деревень
- Парсинг команд формата: `capx4 6515[1];74000[2] 90618`
- Интеграция с прокси-сервером

### 2. C# Прокси-сервер
- HTTP API на порту 5000
- Endpoint `/api/village` - добавление/получение координат
- Endpoint `/api/calculate-attacks` - расчет времени атак
- Кэширование координат деревень
- Расчет расстояний и времени в пути

### 3. Калькулятор атак (Python)
- Расчет расстояния между деревнями
- Расчет времени в пути для разных типов войск
- Парсинг команд атак

## 🎯 Как использовать

### Запуск системы:

1. **Запусти прокси-сервер:**
```cmd
dotnet run --project SHKProxy/SHKProxy.csproj
```

2. **Запусти Discord бота:**
```cmd
py bot.py
```

3. **В Discord:**
```
!addvillages
!time capx4 6515[1];74000[2];71593[3];93485[5];45594[4] 90618
```
