# Stronghold Kingdoms Discord Bot

Discord бот для расчета времени атак в игре Stronghold Kingdoms.

## Что работает

✅ Discord бот с командами
✅ Калькулятор времени атак
✅ C# прокси-сервер для получения координат
✅ Интеграция бота с прокси

## Быстрый старт

### 1. Запусти прокси-сервер (C#)

```cmd
dotnet run --project SHKProxy/SHKProxy.csproj
```

Прокси будет доступен на http://localhost:5000

### 2. Добавь координаты деревень

Через API прокси:
```cmd
Invoke-WebRequest -Uri http://localhost:5000/api/village -Method POST -ContentType "application/json" -Body '{"id": 6515, "x": 100, "y": 200, "name": "Village 1"}' -UseBasicParsing
```

### 3. Запусти Discord бота

```cmd
py bot.py
```

### 4. Используй команды в Discord

```
!time capx4 6515[1];74000[2];71593[3] 90618
```

Бот автоматически получит координаты с прокси и рассчитает время!

## Команды бота

- `!time <тип> <атакующие> <цель>` - рассчитать время атак
- `!coords <id> <x> <y>` - добавить координаты (локально)
- `!ping` - проверка работы

## Следующие шаги

Сейчас координаты добавляются вручную. Для автоматического получения нужно:

1. Интегрировать RemoteServices из игры в прокси
2. Авторизоваться на сервере игры
3. Получать координаты всех деревень автоматически

Прокси-сервер готов к интеграции - нужно только добавить код подключения к игре.
