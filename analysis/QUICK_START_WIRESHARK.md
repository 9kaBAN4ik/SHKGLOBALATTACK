# Быстрый старт: Анализ трафика

## Что нужно сделать (5 минут)

### 1. Скачать и установить Wireshark
- Ссылка: https://www.wireshark.org/download.html
- Выбрать Windows x64 Installer
- Установить со всеми компонентами

### 2. Узнать IP адрес сервера
```bash
ping shk-w756.elb.fireflyops.com
```
Запомни IP адрес (например: 52.18.136.227)

### 3. Запустить Wireshark
1. Открыть Wireshark
2. Выбрать сетевой адаптер (обычно Ethernet или Wi-Fi)
3. В поле фильтра вверху написать:
   ```
   tcp.port == 80 and ip.dst == 52.18.136.227
   ```
   (замени IP на тот, что получил из ping)
4. Нажать синюю кнопку "Start" (акула)

### 4. Запустить игру
1. Открыть Stronghold Kingdoms
2. Войти в аккаунт
3. Выбрать Europa 10
4. Дождаться загрузки мира
5. Подождать 10 секунд

### 5. Остановить захват
1. Вернуться в Wireshark
2. Нажать красную кнопку "Stop"

### 6. Найти нужный пакет
1. В списке пакетов найти строки с `POST /KingdomsRPC/Kingdoms.rem`
2. Кликнуть на первый такой пакет
3. Внизу в панели "Packet Bytes" будут данные

### 7. Экспортировать данные

**Вариант A (проще):**
1. Правой кнопкой на пакет
2. Follow → TCP Stream
3. Скопировать весь текст
4. Вставить в файл `tcp_stream.txt`

**Вариант B (точнее):**
1. Правой кнопкой на пакет
2. Export Packet Bytes
3. Сохранить как `request.bin` в папку проекта

### 8. Анализ
```bash
py analyze_capture.py
```

Скрипт покажет:
- Размер запроса
- Сжатие
- Найденные строки
- GUID'ы
- Параметры

## Что искать в результатах

Сравни с нашими параметрами:
- **userName**: `kara.bridges1991@comejoinuspro.org` - совпадает?
- **userGuid**: `5dbaf4019e9046409f8feb3b55c10137` - совпадает?
- **sessionGuid**: (меняется каждый раз) - есть в захвате?
- **needVillageData**: `true` - совпадает?
- **versionID**: `1` - совпадает?

## Если что-то не работает

### Wireshark не показывает пакеты
- Проверь, что выбран правильный сетевой адаптер
- Попробуй фильтр: `tcp.port == 80`
- Или вообще без фильтра

### Не можешь найти LoginUserGuid
- Ищи пакеты размером ~200-300 байт
- Или все пакеты к `shk-w756.elb.fireflyops.com`
- Экспортируй несколько пакетов

### Игра не подключается
- Проверь интернет
- Попробуй другой мир
- Перезапусти игру

## Альтернатива: Fiddler

Если Wireshark сложен:
1. Скачать Fiddler: https://www.telerik.com/fiddler
2. Запустить Fiddler
3. Запустить игру
4. В Fiddler найти запросы к серверу
5. Посмотреть содержимое

После анализа мы узнаем точные параметры и сможем исправить код!

Тогда бот сможет автоматически получать координаты всех деревень с сервера.
