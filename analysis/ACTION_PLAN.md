# План действий: Получение координат деревень

## Текущая ситуация

✅ CustomSinks.dll исправлен - запросы доходят до сервера
❌ Сервер возвращает ошибку - нужно найти правильные параметры

## Два подхода

### Подход 1: Анализ трафика (ТОЧНЫЙ)

**Время:** 10-15 минут  
**Сложность:** Средняя  
**Результат:** 100% гарантия решения

**Шаги:**

1. **Установить Wireshark**
   ```
   https://www.wireshark.org/download.html
   ```

2. **Узнать IP сервера**
   ```bash
   ping shk-w756.elb.fireflyops.com
   ```

3. **Запустить захват**
   - Открыть Wireshark
   - Выбрать сетевой адаптер
   - Фильтр: `tcp.port == 80 and ip.dst == [IP_АДРЕС]`
   - Нажать Start

4. **Запустить игру**
   - Войти в аккаунт
   - Выбрать Europa 10
   - Дождаться загрузки

5. **Остановить захват**
   - Вернуться в Wireshark
   - Нажать Stop

6. **Экспортировать данные**
   - Найти пакет с `POST /KingdomsRPC/Kingdoms.rem`
   - Правой кнопкой → Export Packet Bytes
   - Сохранить как `request.bin`

7. **Анализ**
   ```bash
   py analyze_capture.py
   ```

8. **Исправить код**
   - Сравнить параметры с нашими
   - Обновить `SimpleGameClient/Program.cs`
   - Перекомпилировать и запустить

**Подробная инструкция:** `QUICK_START_WIRESHARK.md`

---

### Подход 2: Автоматический перебор (БЫСТРЫЙ)

**Время:** 5 минут  
**Сложность:** Низкая  
**Результат:** Может найти решение методом перебора

**Шаги:**

1. **Запустить скрипт**
   ```bash
   py bruteforce_params.py
   ```

2. **Дождаться результата**
   - Скрипт попробует разные комбинации параметров
   - Если найдет рабочую - покажет её
   - Если нет - нужен Подход 1

3. **Если найдена рабочая комбинация**
   - Обновить `SimpleGameClient/Program.cs`
   - Перекомпилировать
   - Запустить

**Преимущества:**
- Не нужен Wireshark
- Автоматический
- Быстрый

**Недостатки:**
- Может не найти решение
- Ограниченный набор комбинаций

---

## Рекомендация

**Начни с Подхода 2** (bruteforce_params.py) - это быстро и просто.

**Если не сработает** - используй Подход 1 (Wireshark) - это гарантированно даст результат.

## После успешного подключения

Когда `LoginUserGuid` заработает:

1. **Получить список деревень**
   ```csharp
   var villages = service.GetVillageNames(userId, sessionId, -1, 0);
   ```

2. **Сохранить в JSON**
   ```csharp
   File.WriteAllText("villages.json", JsonSerializer.Serialize(villages));
   ```

3. **Отправить в прокси-сервер**
   ```csharp
   foreach (var village in villages)
   {
       httpClient.PostAsync("http://localhost:5000/api/village", ...);
   }
   ```

4. **Использовать в Discord боте**
   ```
   !time capx4 12345[1];67890[2] 99999
   ```
