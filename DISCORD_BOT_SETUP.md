# 🤖 Настройка Discord Бота

## Проблема: Токен сбрасывается GitHub

GitHub автоматически сканирует публичные репозитории и сбрасывает Discord токены для безопасности. Поэтому ваш токен больше не работает.

## ✅ Решение: Получить новый токен

### Шаг 1: Получить новый токен Discord

1. Откройте https://discord.com/developers/applications
2. Войдите в свой аккаунт Discord
3. Выберите ваше приложение из списка (или создайте новое)
4. В левом меню выберите **Bot**
5. Нажмите кнопку **Reset Token** (или **View Token**)
6. Подтвердите сброс токена
7. **ВАЖНО:** Скопируйте новый токен сразу - он показывается только один раз!

### Шаг 2: Обновить .env файл

1. Откройте файл `.env` в корне проекта
2. Замените старый токен на новый:
   ```
   BOT_TOKEN = ВАШ_НОВЫЙ_ТОКЕН_ЗДЕСЬ
   ```
3. Сохраните файл

**ВАЖНО:** Файл `.env` теперь в `.gitignore` и НЕ будет коммититься в git!

### Шаг 3: Скопировать .env в папку analysis

```bash
cp .env analysis/.env
```

### Шаг 4: Запустить систему

**Терминал 1 - Прокси сервер:**
```bash
python proxy_server.py
```

**Терминал 2 - Discord бот:**
```bash
cd analysis
python bot.py
```

Или через текущий запущенный проект (прокси уже запущен):
```bash
cd /home/runner/app/analysis
nohup python bot.py > bot.log 2>&1 &
```

### Шаг 5: Проверить статус

```bash
# Проверить, что бот запущен
ps aux | grep "python bot.py"

# Посмотреть логи
tail -f /home/runner/app/analysis/bot.log
```

Если всё хорошо, вы увидите:
```
Бот YourBotName#1234 успешно запущен!
ID: 1234567890
Сервер игры: http://shk-w756.elb.fireflyops.com:80
```

## 🎮 Тестирование

После успешного запуска, протестируйте команду в Discord:

```
/time all fromids: 16107;74469 targetid: 83239
```

Бот должен ответить с таблицей времени атак для всех множителей скорости (x1 - x6).

## 🔒 Безопасность

- ✅ `.env` теперь в `.gitignore`
- ✅ `.env.example` создан с примерами
- ✅ Токен больше не попадёт в git
- ❌ НЕ коммитьте файл `.env` вручную!
- ❌ НЕ делитесь токеном в публичных местах!

## 📝 Создание нового бота (если нужно)

Если у вас нет приложения Discord:

1. Откройте https://discord.com/developers/applications
2. Нажмите **New Application**
3. Введите имя бота (например: "SHK Attack Timer")
4. Перейдите в раздел **Bot** → **Add Bot**
5. Включите опции:
   - ✅ MESSAGE CONTENT INTENT
   - ✅ SERVER MEMBERS INTENT
6. Скопируйте токен
7. В разделе **OAuth2** → **URL Generator**:
   - Выберите scope: `bot`
   - Выберите права: `Send Messages`, `Read Messages`, `Embed Links`
   - Скопируйте сгенерированную ссылку
8. Откройте ссылку в браузере и добавьте бота на ваш сервер Discord
